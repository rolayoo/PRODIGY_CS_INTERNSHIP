#  Prodigy InfoTech Task 01: Caesar Cipher

##  Description
This project is a Python-based interactive implementation of the classic Caesar Cipher. It allows users to encrypt and decrypt text by shifting the letters of the alphabet by a user-defined number of positions. Developed as part of the Cybersecurity Internship at Prodigy InfoTech.

##  Features
* **Encryption:** Secures readable text into a ciphered format.
* **Decryption:** Reverts ciphered text back to its original readable format.
* **Case Preservation:** Maintains the uppercase and lowercase structure of the original message.
* **Non-Alpha Character Handling:** Leaves numbers, symbols, and spaces untouched during the shift.
